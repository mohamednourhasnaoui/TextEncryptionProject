Project Explanation

How the System Works

This section explains the main cryptographic ideas used in the project.

01

Fernet Symmetric Encryption

Fernet uses the same key to encrypt and decrypt a message. The user must keep this key safe because anyone with the key can decrypt the text.

02

RSA Asymmetric Encryption

RSA uses two keys. The public key is used to encrypt the message, while the private key is used to decrypt it.

03

Randomized Encryption Output

Strong encryption does not always produce the same encrypted result. Even if the same message is encrypted twice, the outputs are different.
